1. What is Semantic HTML? 
A. Semantic HTML is the use of HTML markup to reinforce the semantics, or meaning, of the information in webpages and web applications rather than merely to define its presentation or look.

2. What is HTML used for? 
A. HTML stands for Hypertext Markup Language. It allows the user to create and structure sections, paragraphs, headings, links, and blockquotes for web pages and applications. 

3. What is an attribute and where do we put it? 
A. HTML attributes provide additional information about HTML elements. Attributes are always specified in the start tag.

4. What is the h1 tag used for? How many times should I use it on a page?
A. Search Engine Optimization (SEO) - Only use one H1 Element on any page

5. Name two tags that have required attributes
A. img and a

6. What do we put in the head of our HTML document? 
A. The <head> element is a container for <title>, <meta>, <link>, <style> and more

7. What is an id? 
A. The id attribute is a unique identifier which is used to specify the document. It is used by CSS and JavaScript to perform a certain task for a unique element.

8. What elements can I add an id to? 
A. An element with a unique id.

9. How many times can I use the same id on a page? 
A. Once
    
10. What is a class? 
A. A class name can be used by multiple HTML elements.
    
11. What elements can I add a class to? 
A. The class attribute can be used on any HTML element.
    
12. How many times can I use the same class on a page? 
A. There is no technical limit.
    
13. How do I get my link to open in a new tab?
A. Set the target attribute to "_blank"
    
14. What is the alt attribute in the image tag used for? 
A. If the image cannot be displayed. The alt attribute provides alternative information for an image if a user for some reason cannot view it
    
15. How do I reference an id?
A. With a specific id, write a hash (#) character, followed by the id of the element.

    16. What is the difference between a section and a div
A. The <section> tag creates independent sections within a webpage having logically connected content. And the <div> tag is an empty container specifying a division or a section.

    17. What is CSS used for? 
A. CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

Stretch Questions

    1. What is the difference between an inline element and a block element?
A. A block-level element always starts on a new line and takes up the full width available - An inline element does not start on a new line and it only takes up as much width as necessary.

    2. What happens when an element is positioned absolutely? 
A. An element with position: absolute; is positioned relative to the nearest positioned ancestor 

    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
A. display: inline, width: ?px;
