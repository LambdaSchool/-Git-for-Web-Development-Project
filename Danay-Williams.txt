 1. What is Semantic HTML? 
    Semantic HTML is HTML that has meaning. Just by looking at the tags, you can tell what they are there for.
2. What is HTML used for? 
    HTML is used to put content on a page.
3. What is an attribute and where do we put it?
    An attribute is an element that is not textually based, like a link (a), and is put directly in a tag.
4. What is the h1 tag used for? How many times should I use it on a page?
    The h1 tag is used to denote something that is most important on a page, like a title, so it should probably be used once
5. Name two tags that have required attributes
    Anchor tags and image tags.
6. What do we put in the head of our HTML document?
    Things that we don't necessarily want on the computer screen such as fonts and titles.
7. What is an id?
    An id is an attribute that can only be assigned to one element.
8. What elements can I add an id to? 
    ID's can be added to block-level elements.
9. How many times can I use the same id on a page?
    Once.
10. What is a class? 
    A class is an attribute that can be used over and over in many elements
11. What elements can I add a class to? 
    Classes can be added to block-level elements.
12. How many times can I use the same class on a page? 
    Many times.
13. How do I get my link to open in a new tab?
    Right click + "Open link in new tab"
14. What is the alt attribute used for? 
    To denote alternate text for an image in case the original one cannot be shown
15. How do I reference an id?
    Reference an ID in CSS by using the id selector (#)
16. What is the difference between a section and a div
    Divs are block-level elements, sections are in-line elements
17. What is CSS used for? 
    CSS is used to add style to the content on a page
18. How to we select an element? Example - every h2 on the page
    h2{
        insert style changes here...
    }
19. What is the difference between a class and an id? - Give me an example of when I might use each one
    An id can only be used once on a page while classes can be used many times. ID's may be used to assign a value to someone's password but a class would be used for people's home state.
20. How do we select classes in CSS?
    Select classes with a dot. Ex:
    In HTML: <div class="lastName"></div>
    In CSS: .lastName{}
21. How do we select a p element with a single class of “human””?
    .human p{}
22. What is a parent child selector? When would this be useful?
    A parent child selector selects the last child element in a parent and is useful when we don't know how many children are in a parent element.
23. How do you select all links within a div with the class of sidebar?
    .sidebar a{}
24. What is a pseudo selector?
    A pseudo selector is a selector for an element that may not have a set value like the last-child selector in a parent element.
25. What do we use the change the spacing between lines?
    Line-height property
26. What do we use to change the spacing between letters?
    Letter-spacing property
27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
    Text-transform property, assign it to capitalize.
28. How do I add a 1px border around my div that is dotted and black?
    div {
        border: 1px dotted #000000;
    }
29. How do I select everything on the page? 
    With the wildcard (*) selector
30. How do I write a comment in CSS?
    Wrap your comment like so: <!-- Your comment here -->
31. How do I find out what file I am in, when I am using the command line?
    By entering "dir"
32. Using the command line - how do I see a list of files/folders in my current folder?
    By entering "ls"
33. How do I remove a file via the command line? Why do I have to be careful with this?
    By entering "rm" but you have to be careful because once the file is removed, this action can't be undone
34. Why should I use version control? 
    Version control keeps track of and saves changes that are made to a file and allows many people to work on a project at the same time.
35. How often should I commit to github?
    Whenever you add something

    git push

Stretch Questions

    1. What is the difference between an inline element and a block element?
        A block element starts a new paragraph, inline elements do not.
    2. What happens when an element is positioned absolutely? 
        The element can be positioned anywhere on a page because the top, left, right and bottom attributes are not defined by the positioning of other elements
    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width?
        Set the property of the element to relative 
    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
        Display block: paragraph <p>, address <address>, navigation <nav>
        Display inline: Link <a>, image <img>
        Display inline-block: Button <button>
    5. In your own words, explain the box model. What is the fix for the box model? 
        The box model describes how elements are positioned on a computer screen and consists of the content, padding, border and margin. If the box is supposed to be a certain size, then all layers of the box must be measured mathematically to fit that size.
