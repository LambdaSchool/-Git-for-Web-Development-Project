1. What is Semantic HTML?
It's the use of HTML to create a clear meaning of the layout of a website, primarily for search engines, bots, etc.


2. What is HTML used for?
HTML is the markup language for websites that allows us to create a layout and structure of a website.

3. What is an attribute and where do we put it?
Attributes are certain words used inside an opening tag that perform a specific action, or otherwise affect the element.

4. What is the h1 tag used for? How many times should I use it on a page?
An H1 is a header tag used to display information such as "Welcome to Lambda School", and would be the first visible item on a website. Ideally, only one H1 tag should be used.

5. Name two tags that have required attributes
<img>, <a> 

6. What do we put in the head of our HTML document?
doctype, title, links to external files (css, javascript, etc.), meta tags, and scripts. 

7. What is an id?
A unique id for an element.

8. What elements can I add an id to?
Any element.

9. How many times can I use the same id on a page?
Once.

10. What is a class?
Class is an attribute. 

11. What elements can I add a class to?
Any element.

12. How many times can I use the same class on a page?
Multiple times.

13. How do I get my link to open in a new tab?
target="_blank"

14. What is the alt attribute in the image tag used for?
Use to identify what the image is for screen readers and search engine bots (they can't "see" images yet). 

15. How do I reference an id?

16. What is the difference between a section and a div
A <section> means there is content within, whereas <div> doesn't mean anything to a computer.

17. What is CSS used for?
CSS is used to style a web page.

18. How to we select an element? Example - every h2 on the page

19. What is the difference between a class and an id? - Give me an example of when I might use each one
An ID is unique, whereas a class is not. An ID would be used on a single item like navigation, whereas a CLASS can be used on multiple items, like H1-H6 tags, or P tags, etc.

21. How do we select a p element with a single class of “human””?

22. What is a parent child selector? When would this be useful?
An element within an element. An <h2> and <p> tag inside of a <div> tag.

23. How do you select all links within a div with the class of sidebar?

24. What is a pseudo selector?
It's a selector that selects elements that are in a specific state.

34. Why should I use version control?
When working on a team, version control allows every team member to work on the project at the same time. Without version control, this would be a mess.

35. How often should I commit to github?
You should commit to github when you make a fix, or a change.

36. What is the command we would use to push our repo up to github?
git push -u origin branch-name

37. Walk me through Lambda's git flow. 
1. Fork repo (create your own version of a repo)
2. Clone the repo (copy the repo onto your local computer)
3. Create a branch (to work on your own copy of the main files)
4. Edit existing files, or Add new files.
5. Commit changes (edits, adding files).
6. Create a pull-request to submit work back to the primary repo.
