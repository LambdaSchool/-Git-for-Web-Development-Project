    1. What is Semantic HTML?
        Semantic HTML are the elements that explicitly tell you what they do or what they are. For example, title, header, footer and article. 
    2. What is HTML used for?
        HTML is used to structure the webpage and the content within it.
    3. What is an attribute and where do we put it?

    4. What is the h1 tag used for? How many times should I use it on a page?

        An id is a unique word or number that is given to a specific tag and should not be repeated.
    8. What elements can I add an id to?
        You can add an id to any of the HTML elements.

    10. What is a class?
        A class is the name given to an element to be later called in the Cascade Style Sheet.
    11. What elements can I add a class to?
        You can add a class to any of the HTML elements. Also, this is case sensitive.
    12. How many times can I use the same class on a page?
        As many times as you desire, though it is good practice to use it in similar elements that work together with each other.

    14. What is the alt attribute in the image tag used for?
        The alt attribute it is used in case the image doesn't display, it will give information or a description of the image that is supposed to display.
    15. How do I reference an id?
        To reference an id in CSS, you have to call it with the hash character in example "#id".

        A section is used to tie a whole group of headings or titles together when they complement each other in similarities. For example, if 3 or 5 headings talk about car brands but the 6th heading talks about car paints then the 6th heading should be in another section.
    17. What is CSS used for?
        Cascading Style Sheets is used to customize a whole webpage into whetever you desire for it to be pleasing to the eyes.
