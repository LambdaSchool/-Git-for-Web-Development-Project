    1. What is Semantic HTML?

    2. What is HTML used for? 

HTML: Hypertext Markup Language. Creates and structures sections, paragraphs, headings, and links for web pages and apps.

Attributes give addtl. info about elements. They are always in the start tag.

h1 usually defines the most important heading. It should typically only be used once.

the <a> tag and the <img> tag.

<title>, <style>, <meta>, <link>, <script>, and <base>.

An id selector uses the id attribute of an html element to select a specific element.

    8. What elements can I add an id to?

    9. How many times can I use the same id on a page?

It provided alternative information if the a user cannot view an image.

    15. How do I reference an id?

    16. What is the difference between a section and a div

section is semantic and represents a grouping. div is non-semantic and does not represent anything.

    17. What is CSS used for? 

CSS describes the presentation of web pages. Examples include design, layout, and other variations of display.

    18. How to we select an element? Example - every h2 on the page

    19. What is the difference between a class and an id? - Give me an example of when I might use each one

.class is used to specify one or more elements for styling. #id is used to specify a specific element for styling.
.class, for when you want to style multiple headers the same color. #id, for when you want to style all but 1 header the same color.
