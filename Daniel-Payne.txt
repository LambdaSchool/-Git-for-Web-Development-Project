	1. What is Semantic HTML? 
	Semantic HTML is a way to explain what your webpage elements are for ie Header is the header at the top, footer is for items at the bottom.

    2. What is HTML used for? 
	HTML is the language used to display information in broswers.

    3. What is an attribute and where do we put it?
	Attributes are special descriptors for how or what a tag should do. You put them in the start of a tag. [IE: <img src=""> <--src being the attribute for the tag.]

    4. What is the h1 tag used for? How many times should I use it on a page?
	The h1 tag is used for the main header of the page. The content that you really want to set the subject of the rest of the page.

    5. Name two tags that have required attributes input, textarea

    6. What do we put in the head of our HTML document? 
	head is for metadata and other information about your HTML document, including any linked script|css files or fonts

    7. What is an id? 
	'Id' tag is a specialized tag that can be used on an element to anchor it to be called once on your page. 

    8. What elements can I add an id to? 
	All elements

    9. How many times can I use the same id on a page? 
	Once

    10. What is a class? 
	A class is a designation for an element on a page. You may use an element as many times as needed on a page

    11. What elements can I add a class to? 
	All elements

    12. How many times can I use the same class on a page? 
	As many times as you'd like.

    16. What is the difference between a section and a div
	A section is a divider of parts of your webpage used to seperate headers, main contents, titles etc. Div is used to seperate parts of your webpage. 

    17. What is CSS used for?
 	CSS is used to stylize your webpage and its elements.

    18. How to we select an element? Example - every h2 on the page
	h2{} or class --> .<className> or universal selector if wanted to change all elements --> *{}

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
	class is used for all elements of a certain type or class {all footer links we want blue} an id is for one specific instance {I need only the link of 'info' to always be red}
