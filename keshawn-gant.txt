1. What is Semantic HTML? 
It is HTML that gives meaning to our markup.

    2. What is HTML used for? 
    It's a building block, the structure of web content.

    3. What is an attribute and where do we put it? 
    Value to adjust elements. Attributes follow after element, left - right.

    4. What is the h1 tag used for? How many times should I use it on a page?
    H1 is used for the most important section of a site, and should only be used once per page.

    5. Name two tags that have required attributes
    <img src= "image file.jpg" alt= "description of image">

    6. What do we put in the head of our HTML document? 
    <header></header> , <nav></nav>

    7. What is an id? 
    Id is an identifier.

    8. What elements can I add an id to? 
    Unique element only.

    9. How many times can I use the same id on a page? 
    1 per page.

    10. What is a class? 
    Class selector.

    11. What elements can I add a class to? 
    Any element.

    12. How many times can I use the same class on a page? 
    Technically as many as you like, but would recommend a balance.

    13. How do I get my link to open in a new tab?
    <a href="" target="_blank">

    14. What is the alt attribute in the image tag used for? 
    Backup just in case image isn't available, it describes what was once there.

    15. How do I reference an id?
    via css #IdName .

    16. What is the difference between a section and a div
    Section tells the browser in a descriptive way that a section exist, div is more or less just for design purposes.

    17. What is CSS used for? 
    Styles the webpage.

    18. How to we select an element? Example - every h2 on the page
    via css h2{}

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
    Id is most specific and should only be used once per page for something important, while classes are less specific and can be used many times over.
    Example I would use a class for duplicate elements ex: h2, h1, p etc, while an id would be placed on a specific element ex: <header> <footer>.

    20. How do we select classes in CSS?
    via CSS  .ClassName {}

    21. How do we select a p element with a single class of “human””?
    .human p {}

    22. What is a parent child selector? When would this be useful? 
    When you specify the element within a group that you want selected. Perfect when you want specific elements altered. 

    23. How do you select all links within a div with the class of sidebar?
    .sidebar div {}

    24. What is a pseudo selector?
    Special state selected elements.

    25. What do we use the change the spacing between lines?
