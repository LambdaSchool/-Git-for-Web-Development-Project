    Semantic HTML is a portion that pretties up HTML and makes it more legible with a better design.

    7. What is an id? 

    An ID is a element selector in CSS. It only describes/styles one particular element. Can only be used once on a page.

    8. What elements can I add an id to? 

    ID can be added to any one element.

    9. How many times can I use the same id on a page? 

    ID can only be used 1 time on a page.

    10. What is a class? 

    A class is another CSS selector that describes a group of elements together.

    11. What elements can I add a class to? 

    A class can be added to any and every element on a page.

    12. How many times can I use the same class on a page? 

    The same class can be used multiple times on a page.

    You use the A tag, the anchor tag. <a href="url">link text</a>

    It places text where the image is in case a user can't load the image when viewing the Web site.
