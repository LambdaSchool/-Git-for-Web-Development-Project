2. What is HTML used for? 
-) To display content on web

-) a property to define characteristics of an HTML element, it is placed inside the element's opening tag

-) h1 tag is usually used for the main heading of a page

-) it provides an alternate text for an image if the image cannot be displayed

15. How do I reference an id?
-) with the '#' symbol

16. What is the difference between a section and a div
-) section is semantic tag which group the content within, while div is only presentational tag that does not convey any meaning

17. What is CSS used for? 
-) for html styling

18. How to we select an element? Example - every h2 on the page
-) by specifying the tag name, in this case, h2

19. What is the difference between a class and an id? - Give me an example of when I might use each one
-) a class can be used by more than one element where an id can only be used by one element
   example of id: <section id='intro'>, this case, there is only one section with the id of 'intro'
   example of class: <p class='content'>, this case, we can have more than one paragraph elements that share the same class

20. How do we select classes in CSS?
-) by using the dot (.) sign

21. How do we select a p element with a single class of “human””?
-) p.human

22. What is a parent child selector? When would this be useful? 
-) a selector that selects all elements that are a direct child of the parent element
   it is useful when we need a css inheritance so that we can share the parent element's properties

23. How do you select all links within a div with the class of sidebar?

24. What is a pseudo selector?
-) pseudo selector is used when you may not have a set amount of elements, but you still want to style them individually

-) Student will create OWN version of Repo (Fork)
   Student will add TL as a collaborator
   Student then will Clone Repo
   Student will then create a branch (on which the student will implement the project) git checkout -b 'firstName-lastName'
   Student will then submit their PR into the new branch.
   Student will use their own student fork as the base (compare across forks).
   Student will add their TL as a reviewer on the PR
