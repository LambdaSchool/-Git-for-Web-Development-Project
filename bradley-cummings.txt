    19. What is the difference between a class and an id? - Give me an example of when I might use each one

The main difference is specificity and how many elements each styling can be applied.  Class can be applied to multiple elements while id can only be applied to one on a page.  

Example: class
<div class="city">
  <h2>London</h2>
    <p>London is a great city to visit</p>
</div>

<div class="city">
  <h2>New York</h2>
    <p>New York is a great city to visit.</p>
</div>

Example: id
<h1 id="myHeader">My Header</h1>

    22. What is a parent child selector? When would this be useful? 

The parent child selector selects all elements that are a direct child of a specified element.  This is useful in that it allows us to style multiple elements related to the parent in the same fashion.

    23. How do you select all links within a div with the class of sidebar?

    24. What is a pseudo selector?

These are CSS selectors with a colon preceding them.  They are used to show elements in a specific state.  Example:  element that has been clicked/hovered upon.

1. Fork the project in GitHub by clicking “Fork”

2. Clone the project onto your computer
* Press green “Code” button
* HTTPS
* *Copy the URL*
* Go to terminal
*  cd Desktop
* [Desktop] mkdir lambdaProjects
* [Desktop] cd lambdaProjects
* [lambdaProjects] git clone *paste URL*

3. Make a branch
* [lambdaProjects] cd “file name"
* [“file name”] git checkout -b user-name

4. Open in VSCode
* “file name” git:  code .
* if that command doesn’t work, just open your project in VS Code

5. While coding commands to enter into Terminal (source)
* git status (shows you what has been modified)
* git add . (adds all of the changes that you’ve made in any file)
* git commit -m “message” 
* git push -u origin “user-name"
