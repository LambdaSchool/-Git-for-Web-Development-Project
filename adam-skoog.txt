1. What is Semantic HTML? 
Semantic HTML gives meaning to a web page through tags like <header><section>.

2. What is HTML used for? 
HTML provides the foundational structure for web pages.

3. What is an attribute and where do we put it? 
Attributes provide additional information to elements and we can nest them inside elements.

4. What is the h1 tag used for? How many times should I use it on a page?
h1 is the most important header tag and you should only use it once.

5. Name two tags that have required attributes
<img> has src and <a> has href

6. What do we put in the head of our HTML document? 
Think of the head like your own head, it keeps all the important information that you don't want to present to public. 
So we can reference style sheets and other things that the end user doesn't need to see.

7. What is an id? 
You can add a unique id to html elements. You can reference an id in CSS to style it and use it to direct users to a certain section.
<section id="call-to-action">

8. What elements can I add an id to? 
It can be added to any HTML element.

9. How many times can I use the same id on a page? 
They should be unique on a webpage.

10. What is a class? 
Elements can have the same class name to apply the same styles to them.

11. What elements can I add a class to? 
Any HTML element.

12. How many times can I use the same class on a page? 
As much as you would like.

13. How do I get my link to open in a new tab?
target="_blank"

14. What is the alt attribute used for? 
Used for browsers that are not able to render certain things on a webpage like an image and provides alternative data

15. How do I reference an id?
With a #

16. What is the difference between a section and a div
Divs contain no meaning unless assigned attributes, whereas sections keep the content grouped.

17. What is CSS used for? 
CSS is used to style HTML elements, ids and classes

24. What is a pseudo selector?
They specify a specific state of an element example is :hover. If a user hovers over an element you can then make new style changes.
